Install software

Manage, look, explore, uninstall, install and update software with the new FreeBSD package manager called Pkgng. Just type "pkg search" plus what you're looking for, To install type "pkg install" plus the name you want to install. Pkgng is a powerful package manager that will update software dependencies when installing new software.
